### Community Support

- **GitHub Issues**: For bug reports and feature requests, please use [GitHub Issues](https://github.com/VoxHash/HandForge/issues)
- **Email**: For direct support, contact contact@voxhash.dev

### Before Asking for Help

1. Check the [FAQ](docs/faq.md) for common questions
2. Review the [Troubleshooting Guide](docs/troubleshooting.md)
3. Search existing [GitHub Issues](https://github.com/VoxHash/HandForge/issues) for similar problems
4. Ensure you're using the latest version of HandForge
5. Verify FFmpeg is installed and working correctly

### Reporting Issues

When reporting an issue, please include:

- HandForge version
- Operating system and version
- Python version
- FFmpeg version
- Steps to reproduce the issue
- Expected behavior
- Actual behavior
- Error messages (if any)
- Screenshots (if applicable)
